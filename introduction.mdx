# Welcome to Theme Raed

Theme Raed is the starting point for developers to design themes for merchant stores that reflect the uniqueness of each store on the Salla Platform. Custom themes make it easier for developers to tailor the merchant's store to its changing demands over time.

## What is Theme Raed?

Theme Raed is shipped as the default theme along with the Twilight Themes engine, which is Salla's powerful theming system for creating customizable e-commerce themes for the Salla Platform.

## Theme features

Theme Raed comes with a comprehensive set of pre-defined components that you can configure through the `twilight.json` file:

- **Youtube** - Display YouTube videos on your store
- **Fixed Banner** - Showcase important banners in a fixed position
- **Testimonials** - Display customer reviews and testimonials
- **Parallax background** - Create engaging 2D parallax effects
- **Photos slider** - Image carousel for product galleries
- **Store Features** - Highlight store capabilities and benefits
- **Square photos** - Display images in uniform square layouts
- **Fixed products** - Pin specific products to always display
- **Products slider** - Navigate through products horizontally or vertically
- **Featured products** - Three distinct styles for showcasing featured items
- **Latest Products** - Automatically display newest additions
- **Vertical menu with slider** - Combined navigation and content slider

## Custom components

In addition to theme features, Theme Raed includes customizable components that you can easily modify:

- **Brands** - Display brand logos in an attractive layout
- **Enhanced Squared Image** - Advanced square image display with animations
- **Main links** - Quick navigation links to important pages
- **Enhanced Slider** - Advanced slider with custom animations and overlays
- **Slider products with header** - Product carousel with customizable headers
