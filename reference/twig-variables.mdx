Every Twig template has access to global variables and page-specific variables.

## Global variables

These variables are available in all templates.

#### Theme colors

<CodeGroup>
```twig Color variables
{{ theme.color.primary }}
{# Primary brand color #}

{{ theme.color.text }}
{# Text color (#000000 or #FFFFFF based on primary) #}

{{ theme.color.reverse_primary }}
{# Reverse of primary color #}

{{ theme.color.reverse_text }}
{# Reverse of text color #}

{{ theme.color.is_dark }}
{# Boolean: is primary color dark? #}
```
</CodeGroup>

#### Theme color functions

<CodeGroup>
```twig Darker shade
{{ theme.color.darker(0.15) }}
{# 15% darker than primary #}

{{ theme.color.darker(0.25, '#ff6767') }}
{# 25% darker than specified color #}
```

```twig Lighter shade
{{ theme.color.lighter(0.15) }}
{# 15% lighter than primary #}

{{ theme.color.lighter(0.3, '#414042') }}
{# 30% lighter than specified color #}
```
</CodeGroup>

<CodeGroup>

</CodeGroup>

### Free shipping bar

<CodeGroup>
```twig Free shipping progress
{% if cart.free_shipping_bar %}
  <div class="free-shipping-progress">
    {% set is_free = cart.free_shipping_bar.has_free_shipping %}
    
    {% if is_free %}
      <p>{{ trans('pages.cart.has_free_shipping') }}</p>
    {% else %}
      <p>{{ trans('pages.cart.free_shipping_alert', {
        'amount': cart.free_shipping_bar.remaining|money
      }) }}</p>
      
      <div class="progress-bar" style="width: {{ cart.free_shipping_bar.percent }}%"></div>
    {% endif %}
  </div>
{% endif %}
```
</CodeGroup>

<ParamField path="cart.free_shipping_bar.minimum_amount" type="float">
  Minimum cart total for free shipping
</ParamField>

<ParamField path="cart.free_shipping_bar.has_free_shipping" type="bool">
  Free shipping threshold reached
</ParamField>

<ParamField path="cart.free_shipping_bar.percent" type="float">
  Progress percentage (0-100)
</ParamField>

<ParamField path="cart.free_shipping_bar.remaining" type="float">
  Amount remaining to reach free shipping
</ParamField>

## Helper functions

### link

Generates store URLs.

```twig
<a href="{{ link('/') }}">Home</a>
<a href="{{ link('/cart') }}">Cart</a>
```

### is_page

Checks current page.

```twig
{% if is_page('index') %}
  <h1>{{ store.name }}</h1>
{% else %}
  <h2>{{ store.name }}</h2>
{% endif %}
```

### trans

Translates text.

```twig
{{ trans('pages.cart.empty_cart') }}
{{ trans('common.elements.back_home') }}
```
